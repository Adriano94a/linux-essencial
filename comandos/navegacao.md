# 📂 Navegação no Sistema de Arquivos Linux

## 🔹 Comandos principais:
| Comando | Descrição |
|---------|-----------|
| pwd     | Mostra o diretório atual |
| ls      | Lista arquivos e pastas |
| cd      | Navega entre diretórios |
| mkdir   | Cria pastas |
| touch   | Cria arquivos |

## 🧠 Exemplos práticos:
```bash
pwd
cd Documentos
mkdir projetos
cd projetos
touch arquivo.txt
ls -l
```
